Напишите функции и тесты к ним. Для каждой функции должно быть не менее
8 тестов.Используйте 2 файла: functions.py и test_functions.py
````
функцию calculate_sum(n), возвращающую сумму чисел от 1 до n;
Пример:
print(calculate_sum(3)) -> 6

функцию count_words(line), возвращающую количество слов в строке (слова разделеныпробелами);
Пример:
print(count_words("Hello world! Goodbye")) -> 3

функцию
is_number(string), которая проверяет, что переданная строка это целое число;
Пример:
print(is_number("123")) -> True

print(is_number("abc")) -> False

функцию unique(lst), возвращающую список уникальных элементов в переданном списке.
Пример:
print(unique([1 , 2 , 2 , 3 , 3])) -> [1, 2, 3]
````
